# Insurance Purchase Prediction

## 📌 Project Overview
This project predicts whether a customer will purchase insurance using machine learning.

## ⚙️ Models Used
- Logistic Regression
- Random Forest
- XGBoost (Final Model)

## 📊 Final Results
- ROC-AUC: 0.62
- Recall: 44%
- Precision: 5.8%

## 🎯 Key Features
- Handled imbalanced dataset
- Applied threshold tuning
- Used XGBoost for best performance
